# registration_flask

Overview

I maintain this repository `registration_flask` which contains an example app for user registration built with Flask. This README explains how to install and run the app.

Features

- User registration and authentication examples
- Form handling and validation
- Simple database integration

Tech stack

- Python
- Flask

Installation

1. Clone the repository:

```bash
git clone https://github.com/idelzss/registration_flask.git
cd registration_flask
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

Usage

Run the app locally:

```bash
export FLASK_APP=app.py
flask run --reload
```

Or:

```bash
python app.py
```

Development

1. Create a virtual environment
2. Install dependencies
3. Run the application and test registration flows
